### Deploy Node.js

* Download nodejs v5.8.0 source code<p>
`curl https://nodejs.org/dist/v5.8.0/node-v5.8.0.tar.gz -o node-v5.8.0.tar.gz`<p>

* Untar<p>
`tar -zxvf node-v5.8.0.tar.gz -C ./`<p>

* Enter the folder<p>
`cd node-v5.8.0/`<p>

* make & install Node.js to your system<p>

        ./configure
        make
        (sudo) make install
